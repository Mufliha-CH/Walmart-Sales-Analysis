# Walmart Sales Analysis Dashboard

## Overview

This project presents an interactive Power BI dashboard built to analyze Walmart sales data and uncover business insights through data visualization.

The dashboard helps identify sales trends, customer purchasing behavior, regional performance, product category analysis, and overall business performance using dynamic Power BI reports.

---

## Objectives

- Analyze overall sales performance
- Identify top-performing products
- Compare regional sales
- Monitor customer purchasing trends
- Track profit and revenue KPIs
- Support data-driven business decisions

---

## Dashboard Features

- Interactive Filters (Slicers)
- KPI Cards
- Sales Trend Analysis
- Product Category Analysis
- Regional Performance
- Customer Insights
- Revenue Analysis
- Profit Analysis
